# Design Assets  AI Prompts & Specifications

## Asset Guidelines
- Format: SVG for icons/illustrations when possible
- Hero images: WebP, max 100KB
- Illustrations: SVG or WebP, max 20KB each
- Consistent palette: evergreen + warm neutrals

## Icon Requirements
- Outline icons for neutral states
- Filled icons for status emphasis
- Always pair with text labels

## Accessibility Notes
- Provide alt text for every asset
- Avoid text embedded inside images
