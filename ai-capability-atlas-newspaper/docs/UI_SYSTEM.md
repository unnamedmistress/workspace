# UI DESIGN SYSTEM

## 3 VISUAL DIRECTIONS
- **Direction A:** Minimalist, calming colors
- **Direction B:** Bold, large typography
- **Direction C:** Professional, subtle animations
- **Chosen:** Direction A - Ideal for clarity on job sites

## TYPOGRAPHY
- Font: Sans-serif
- Sizes: 18px+ body, 24px+ headers
- Line Height: 1.5+

## COMPONENT INVENTORY
- Big buttons (44px tap targets)
- Simple forms
- Progress indicators

## COLOR SYSTEM
- Primary: Trustworthy blues
- Secondary: Action-driven greens
- Success/Warning/Error: Clear meanings

## INTERACTION STATES
- Clear and concise, focused on usability

- Bold Choice: Large permit search button
- Guardrail: Keyboard accessibility ensured
