## Learn New Things: Learn New Things

### üõ†Ô∏è Cognitive Toolkit

- Abstraction

### Explain complex concepts

**What it is:** Breaking down complicated topics into understandable parts using analogies and simple language

**When to use:** When learning something new, teaching others, or encountering unfamiliar terminology

**Cognitive Skills Required:**

- **Foundational:** Pattern recognition, Information retrieval
- **Intermediate:** Abstraction, Analogy formation, Simplification

üí° **Thinking Prompts:**

- How can I chunk this information?
- What prior knowledge does my audience have?

- Using jargon without explanation

ü§î **Metacognitive Check-ins:**

- Is this at the right level?
- What questions might arise?
