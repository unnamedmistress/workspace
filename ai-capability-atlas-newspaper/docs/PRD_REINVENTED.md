# PRODUCT STRATEGY

## TARGET AUDIENCE
- **Persona:** Alex, 40, General Contractor
- **Role:** Manages multiple construction projects
- **Daily Context:** Needs efficient tools for permit acquisitions

## JOBS-TO-BE-DONE
1. Organize and Apply for Permits Efficiently
   - **When:** During project planning and adjustments
   - **Painful Solutions:** Navigating complex municipal websites
   - **Anxieties:** Missing deadlines

2. Ensure Compliance with Local Codes
   - **When:** Before and during project execution
   - **Painful Solutions:** Consulting multiple sources for code updates
   - **Anxieties:** Facing delays or fines

3. Track and Retrieve Permits Instantly
   - **When:** On-site during inspections
   - **Painful Solutions:** Paper trail management
   - **Anxieties:** Misplacing documents

## 3 STRATEGIC PIVOTS
- **Pivot A:** Instant Permit Finder
  - **Differentiation:** Quick permit identification

- **Pivot B:** Compliance Companion
  - **Differentiation:** Alerts for code updates

- **Pivot C:** Smart Permit Hub
  - **Differentiation:** Integrated dashboard

- **Chosen Pivot:** Smart Permit Hub
  - **Justification:** Centralizes information and reduces administration.
