# Routing Rules and Keyword Patterns

## Vision Stage Keywords
- User needs, insights, value proposition

## PRD Stage Keywords
- Specifications, features, requirements

## Architecture Stage Keywords
- System design, architecture, technical decisions

## Build Stage Keywords
- Implementation, coding, development, CRUD

## QA Stage Keywords
- Testing, audits, security reviews, code quality

## Deployment Stage Keywords
- Deployment, CI/CD, infrastructure
